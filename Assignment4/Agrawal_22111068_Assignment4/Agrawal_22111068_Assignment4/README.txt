To run the solution :-
# Command :- (inside the folder) 

python solution.py

After entering the above command it asks :-

ENTER SAMPLING PERCENTAGE (percentage of data you want to be sampled) :

Where you have to enter the percentage od data you want to sample from the original. 


After this it asks :-

ENTER RECONSTRUCTION METHOD (nearest/linear) :

Here, you have to type the method of reconstruction. i.e. type linear or nearest.

Ater this a file named 'sampled_points.vtp' and 'ouput_nearest.vti' or 'output_linear.vti' (depending on the method you chose) will be generated inside the same folder.
which has the randomly selected sample points and the reconstructed dataset respectively. These files can be viewed in paraview.
